# 📊 Student Performance Analyzer

A menu-driven Python application that analyzes student performance using **NumPy** and a real-world **Kaggle CSV dataset**. The project performs statistical analysis, generates reports, and provides insights into student performance through an interactive command-line interface.

---

## 🚀 Features

- 📂 Load student data from a CSV file
- 📋 Display complete dataset
- 🔍 Search student by ID
- 📊 Calculate total and average marks
- 🏆 Find topper and lowest scorer
- 📚 Calculate subject-wise averages
- 📈 Display highest and lowest marks in each subject
- 📝 Generate student performance reports
- 🎓 Assign grades based on percentage
- ✅ Determine pass/fail status
- 🏅 Rank students by performance
- 📉 Display class statistics (Mean, Median, Variance, Standard Deviation)
- 🎯 Filter and compare student records
- 📌 Menu-driven interface for easy navigation

## ▶️ How to Run

1. Clone the repository

```bash
git clone <repository-link>
```

2. Navigate to the project folder

```bash
cd Student-Performance-Analyzer
```

3. Install NumPy

```bash
pip install numpy
```

4. Run the project

```bash
python main.py
```
